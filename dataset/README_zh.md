# Dataset 文件分布

所有的数据都应该放在 `dataset/` 目录下的两个子文件夹中：

- `dataset/data/`: CSV 文件，包含原始传感器信号。
- `dataset/label/`: CSV 文件，包含对应信号文件的标签。

`data` 与 `label` 子文件夹中的文件名称 **必须完全匹配**（例如，`example.csv` 必须同时存在于两个文件夹中）。数据加载器会检查这两个文件夹中文件名的集合是相同的，如果有任何文件在其中一个文件夹中缺失，就会抛出错误。

每一个 CSV 文件都应该 **没有表头行**。数据文件应该包含时间序列值（排列方式使得加载后得到的二维张量表示为通道数 × 时间步长），标签文件应该包含每个时间步长的一个数值，代表 `no_fire` 为 0 或 `fire` 为 1。（注意，标签文件的行数应该与数据文件相同）
